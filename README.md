# lessons-widget-development

公开课题目：GeoScene API for JavaScript中的微件定制及扩展

讲师：张光耀

概述：在地图开发中，有很多常用的功能，被做成了微件，比如底图切换、指北针、图层列表等。使用微件，可以节省大量研发时间。公开课将从微件的使用讲起，并最终实现一个自定义的微件。
